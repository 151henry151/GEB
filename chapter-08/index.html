Chapter VIII: Typographical Number Theory

We can write numbers and "plus" and "times" using only a few symbols: zero, "next" (one more), and signs for add and multiply. So "next next zero" is two, and "next next zero + next next next zero" is 2+3. All of math with numbers can be turned into a game of symbols.

That game is called TNT—Typographical Number Theory. Every rule is just "replace this shape with that shape." But the shapes turn out to be number facts. So the book is writing number math with symbols.

TNT: arithmetic as symbol manipulation

TNT uses symbols: 0 (zero), S (successor: "the next number"), +, ·, =, and variables (a, b, c, …). Terms like S0, SS0 stand for 1, 2. Equations like SS0+SSS0=SSSSS0 say 2+3=5. Axioms and rules let you derive true number facts by symbol pushing alone.

Why it matters for Gödel

Gödel's trick is to assign a number to every formula and proof. So "this formula is not provable" becomes a number fact that a formula of TNT can express. TNT then "talks about" its own formulas—that's the key to the incompleteness theorem.

Typographical Number Theory

Hofstadter presents TNT: a formal system whose language includes 0, S (successor), +, ·, =, variables, and logical connectives and quantifiers. Terms represent natural numbers (S…S0 = n); formulas represent statements about numbers. Axioms and rules (e.g. induction) allow derivation of number-theoretic truths. Every theorem, when interpreted, is a true statement of arithmetic.

Arithmetization of syntax

The stage is set for Gödel numbering: assigning a unique number to each symbol, formula, and proof. Then "being a proof of formula X" becomes a number-theoretic relation. So TNT can encode statements about its own formulas—the precondition for self-reference and incompleteness.
