Chapter X: Levels of Description, and Computer Systems

A computer can be described in many ways: it is adding numbers, or it is moving little switches, or it is running a program. Same machine, different levels. Sometimes one level describes another level.

Multiple levels

Hardware, machine code, high-level program: each is a level of description of the same process. Systems can describe or run other systems.

Ant Fugue and Aunt Hillary

The dialogue introduces levels of thought and the idea that consciousness might involve tangled levels of description.

Levels of description

Hofstadter discusses how the same process can be described at multiple levels: hardware, instruction set, high-level language. Levels can see or implement other levels. This prepares the reader for self-reference and for the idea that minds might be multi-level systems.
