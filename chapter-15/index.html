Chapter XV: Jumping out of the System

Inside the game, the rules never prove that special sentence G. So inside the game we never see "G is true." But we are outside the game. We can see that G says "you cannot prove me," and we already know the rules do not prove it, so what G says is right. So we know G is true even though the game never gets there.

That is jumping out: we use what we know outside the rules to see something the rules themselves cannot show.

Inside TNT

From inside TNT, we only have proofs. G has no proof. So inside the system we cannot establish that G is true; we only see that we have not (and by the construction never will) derive G. So from inside, G is "undecidable."

Outside: we see the truth

We are not inside TNT. We understand what G means: "I am not provable in TNT." And we have shown that TNT does not prove G. So the claim G makes is correct. So we know G is true. We have "jumped out" of the system to see a truth the system cannot prove.

Essential incompleteness and Lucas

The same trick can be repeated: add G as an axiom, and the new system has its own unprovable sentence. So there is no finite way to make TNT complete. The chapter also discusses Lucas's argument (that Gödel shows minds are not mechanical) and why many philosophers reject it: the machine could "jump out" too, or the comparison is flawed.

Seeing the truth of G from outside

Hofstadter emphasizes that although TNT does not prove G, we can recognize that G is true. We do so by understanding the construction: G is equivalent to "G is not provable in TNT," and we have established that TNT does not prove G, so the proposition expressed by G holds. Our reasoning is not carried out inside TNT; it is meta-theoretic. So "jumping out" is moving to a richer perspective (the meta-system) from which we can see the truth of a sentence the object system cannot prove.

Essential incompleteness

If we add G as an axiom to TNT, we get a new system TNT+G. That system also has a Gödel sentence G'. So incompleteness is not fixed by adding one axiom; it is essential. No consistent, sufficiently strong formal system of arithmetic can prove all arithmetic truths.

Lucas and the anti-mechanist argument

The chapter presents and criticizes J. R. Lucas's argument that Gödel's theorem shows the mind cannot be a machine: the idea that we can always "see" the truth of the Gödel sentence, while the machine cannot. Hofstadter (and many others) point out that the machine could be equipped to output its own Gödel sentence as true, or that the comparison between "what we see" and "what the machine does" is not straightforward. So Gödel's theorem does not refute mechanism.
